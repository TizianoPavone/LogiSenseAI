# 🚀 LogiSense AI

LogiSense AI è un software gestionale intelligente per la Supply Chain. Automatizza il calcolo dell'EOQ (Economic Order Quantity), analizza il rischio di stockout tramite modelli di regressione e integra un assistente AI basato su Google Gemini per supportare le decisioni strategiche e la negoziazione con i fornitori.

## 🛠 Funzionalità Principali
- **Gestione Scorte**: Calcolo automatico EOQ per ottimizzare i livelli di magazzino.
- **Stress Test**: Simulazione dinamica di scenari di domanda e ritardi fornitori.
- **Supply Chain Copilot**: Integrazione AI per analisi predittive ed elaborazione di email commerciali.

## 📦 Come Iniziare
1. **Configurazione**: Accedi all'app e carica il tuo file inventario (CSV o Excel).
2. **Setup AI**: Inserisci la tua API Key di Gemini nella barra laterale per sbloccare le funzioni di analisi avanzata.
3. **Analisi**: Naviga tra le schede per visualizzare il piano ordini e interrogare il Copilot.
