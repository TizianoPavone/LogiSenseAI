# 🧠 Configurazione Assistente AI (Gemini)

LogiSense AI integra l'intelligenza artificiale per aiutarti a negoziare con i fornitori e analizzare i dati. Per abilitare queste funzioni, devi collegare la tua chiave personale di Google Gemini.

### Procedura di Configurazione
1. **Genera la tua API Key**:
   - Vai su [Google AI Studio](https://aistudio.google.com/).
   - Accedi con il tuo account Google.
   - Nella barra laterale sinistra, clicca su **"Get API key"**.
   - Clicca su **"Create API key"** (in un nuovo progetto).
2. **Copia la chiave**: Copia la stringa alfanumerica generata.
3. **Attiva nell'App**:
   - Torna sulla pagina di **LogiSense AI**.
   - Nella barra laterale sinistra dell'app, troverai la sezione **"🧠 Configurazione AI"**.
   - Incolla la tua chiave nel campo protetto "Inserisci API Key Gemini".

### Sicurezza e Privacy
* **Sicurezza**: La tua chiave non viene salvata nei nostri server. Viene utilizzata esclusivamente in locale nel tuo browser per comunicare direttamente con Google.
* **Costi**: L'utilizzo tramite il piano gratuito di Google AI Studio non comporta alcun costo e non richiede inserimento di carte di credito.
* **Autonomia**: Ogni utente utilizza la propria chiave, garantendo totale autonomia e privacy sui dati scambiati con l'AI.
